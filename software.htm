Software:

Estos son los programas informáticos que hacen posible la ejecución de tareas específicas dentro de un computador. Por ejemplo, los istemas operativos, aplicaciones, navegadores web, juegos o programas.

- Software de sistama: sistema operativo, los controladores de dispositivos.

-Software de aplicación: programas qye se utiliza para realizar determinada tarea.

-Software de programación: herramientas digitales que sirven para la creación de nuevos programas.
